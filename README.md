# SmartPath

The goal is to help students plan tasks, stay focused, and prioritize important work.

## Key Features

- Intelligent Daily Scheduling
- Proactive AI Assistant
- Smart Task Classification
- Deep Focus Mode (Pomodoro)

## Project Scope (Learning Version)
we focus on core features and avoid overcomplicated functionality.

- Task creation and daily planning
- Priority and urgency classification
- Basic AI guidance for next steps
- Pomodoro timer for focus sessions
